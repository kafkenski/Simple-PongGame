# PongGame

So, I created my first game ever which is pong game. 

I'm pretty happy with the result, but still there's a few bugs.

Controls:

Player 1: W / S

Player 2: ↑ / ↓

The game is played to 3 points, afterwards you can reset the game by pressing SPACE.
